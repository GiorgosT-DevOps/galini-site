# Galini Suites & Café — Official Website

Official website for **Galini Suites & Café**, designed and developed to present the accommodation, café experience, menu offerings, gallery, and contact information through a modern, elegant, and responsive web interface.

The project focuses on **clean hospitality design, refined typography, balanced spacing, and smooth user interactions**, ensuring an enjoyable browsing experience across desktop and mobile devices.

---

# Project Overview

The Galini website is a **custom-built front-end project** created specifically for the needs of Galini Suites & Café.

The website provides visitors with clear information about the property while maintaining a calm and sophisticated digital identity.

The platform includes several core sections:

- Home page introducing the brand and atmosphere
- Menu page presenting food and beverage offerings
- Image gallery showcasing the environment and experience
- Contact page with communication details and inquiry form

The design emphasizes **clarity, elegance, and usability**, reflecting the hospitality character of the business.

---

# Features

## Responsive Design
The website is optimized for:

- Desktop devices
- Tablets
- Mobile screens

Layouts adapt dynamically to ensure readability and visual consistency.

---

## Interactive Gallery

The gallery section includes:

- Grid-based image layout
- Interactive lightbox viewer
- Image navigation
- Zoom functionality
- Smooth open and close animations
- Background dimming for visual focus

---

## Menu Presentation

The menu page organizes items into clear sections such as:

- Brunch
- Lunch
- Drinks
- Cocktails
- Beverages

The layout is designed to maintain **clear visual hierarchy and comfortable reading flow**.

---

## Contact Section

The contact page provides:

- Location details
- Phone and email contact
- Opening hours
- Map access
- Contact form

The design emphasizes **spacious layout and readability**.

---

# Technologies Used

Core technologies used in the development of the website:

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**

Typography:

- **Playfair Display**
- **Montserrat**

The project intentionally avoids external frameworks to maintain **performance, simplicity, and full design control**.
